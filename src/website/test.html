System Features

  • SIGNAL CONTROLLER Automation

    For efficient traffic movement
    Automate the traffic lights without manual control.
  • This proposed system is a 5 step methodology
  • Image Cleaning
  • Feature Extraction
  • Training the model
  • Density Estimation
  • Traffic Light Signalling

  • LINE VIOLATION Violation

    For capturing signal jumps
    Video processing model for identifying signal jumping
  • Model is implemented by drawing the traffic line on the road
  • Line indicates that traffic light is red in state
  • Violation occurs when vehicles cross the traffic line
  • YOLO Algorithm is used for object detection
  • Once the vehicles are detected violation cases are checked
  • Bounding box around the vehicle becomes red after violation

  • ANPR Feature

    Identifying number plate for violated vehicle
    Image processing model for identifying number plate
  • If the helmet is not found then we proceed to this step
  • Bounding box is created across the license plate for the image
  • Json file stores info like coordinates of the bounding box etc
  • Extracted images are stored so that we can
  • Use the coordinates stored in the file to extract the license plate
  • Confidence greater than threshold is chosen for reading

  • HELMET DETECTION Violation

    Identifying the person who not wearing helmet
    Model for detecting non helmet riders
  • The Image is given as input to the helmet detection model
  • If rider is wearing helmet then further processing is not required
  • If rider is not wearing the helmet then process starts
  • We need to crop image and process it to the model
  • Where a bounding box around the helmet with probability
  • Displayed
